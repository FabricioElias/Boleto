load_boleto(Array $arguments) 
Inherited

It first checks if there is an plugin implementation for the desired bank. If the plugin is present than it instantiates and returns the object otherwise it returns a error message.

Check out the test folder to see how to call this method statically which will then instantiate the Boleto object accordingly.

data_vencimento() 
Inherited

Argument values expected are: -1 for Cash against document. An Integer Number which defines the number of days to be added on top of issuing date. A date formated as dd-mm-yyy.

If argument is empty then it adds 5 days on top of issuing date as default.
